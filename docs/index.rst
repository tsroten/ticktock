Welcome to Tick Tock
====================

Tick Tock adds useful features to Python's :class:`Shelf <shelve.Shelf>` class.
This is Tick Tock's documentation. It contains basic and advanced information
about how to use Tick Tock. If you haven't installed Tick Tock yet, try
starting at :doc:`installation`. If you want to get started with a simple
introduction, read :doc:`quickstart`. For more in depth information, check out
the :doc:`api`.
